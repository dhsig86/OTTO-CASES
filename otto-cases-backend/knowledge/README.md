# OTTO CASES - Knowledge Base (Base de Conhecimento)

Esta pasta foi reservada para o "Deep Research" da área médica. 
Salve aqui arquivos em formato texto (`.txt`, `.json`, ou `.md`) contendo os fragmentos de consensos médicos, artigos científicos de Otorrinolaringologia e Cirurgia de Cabeça e Pescoço. 

A "Regra 11" dita que usaremos a **Pipeline de Extração** para alimentar nossa IA. 

## Como usar:
1. Faça sua pesquisa avançada usando ChatGPT ou Gemini.
2. Peça à IA de pesquisa para resumir os guidelines chave no formato de tópicos técnicos.
3. Cole o texto dentro de um arquivo nesta pasta, ex: `sccp_guideline_2025.txt`.
4. Mais adiante nosso backend de inference carregará esses dados no contexto antes de pedir para a IA redigir o relato de caso.
